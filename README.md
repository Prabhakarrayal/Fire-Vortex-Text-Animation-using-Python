## üìñ Project Overview
This project is a **Python-based visual animation system** that creates a cinematic **fire effect** for any user-entered text.

The animation includes a **fire vortex intro**, **glowing fire text formation**, **heatwave distortion**, **smoke simulation**, and a **full fire explosion outro**.

The main objective of this project is to understand how **particle systems, physics-based motion, and real-time rendering** are used in modern animations and game engines.

## üéØ Key Features
- User-input based text animation  
- Fire vortex intro using circular motion  
- Particle-based text formation  
- Glow pulse effect using sine waves  
- Heatwave distortion below the text  
- Smoke particle simulation  
- Fire explosion outro with shockwave  
- Automatic font scaling for long text  
- Smooth 60 FPS real-time rendering
